# 대사/씬 작성 가이드

## 씬 구조

모든 씬은 세 요소를 포함한다:

1. **목표**: 이 씬에서 달성할 것 (정보 전달, 관계 변화, 갈등 심화 등)
2. **갈등**: 목표를 방해하는 것 (대립, 긴장, 장애물)
3. **전환**: 씬 끝에서 상황이 바뀐 것 (새 정보, 결심, 관계 변화)

> **늦게 들어가고 일찍 나오기**: 씬의 시작/끝에서 불필요한 인사, 이동, 전환을 생략한다.

## 묘사 기법

### 오감 활용

시각 편중을 피하고, 장르에 맞는 감각을 강조한다:

| 감각 | 효과 | 적합한 상황 |
|------|------|------------|
| 시각 | 공간감, 외형 | 풍경, 첫 등장 |
| 청각 | 분위기, 긴장 | 전투, 공포, 고요 |
| 촉각 | 친밀감, 고통 | 로맨스, 액션 |
| 후각 | 기억, 장소감 | 회상, 이국적 배경 |
| 미각 | 문화, 일상 | 식사, 독, 축제 |

### Show, Don't Tell

```
❌ "그는 화가 났다."
✅ "주먹이 하얗게 질릴 때까지 쥐어졌고, 턱 근육이 씰룩거렸다."
```

감정을 직접 서술하지 않고 **행동, 신체 반응, 환경 변화**로 표현한다.

### 의미 있는 디테일

묘사할 디테일을 선택할 때:
- 캐릭터성을 드러내는가?
- 테마를 강화하는가?
- 복선을 심는가?
- 위 셋 중 하나도 해당하지 않으면 생략을 고려한다.

## 대사 작성

### 서브텍스트

말하는 것과 의미하는 것의 간극 — 대사의 깊이를 만든다.

```
표면: "괜찮아, 신경 쓰지 마."
이면: (전혀 괜찮지 않지만, 부담을 주고 싶지 않다)
```

### 갈등이 담긴 대화

대화에서 갈등을 드러내려면:
- 두 캐릭터가 **서로 다른 목표**를 갖고 대화에 임하게 한다
- 직접 충돌하지 않아도, 각자의 의도가 엇갈리면 긴장이 생긴다

### 정보 전달 (Exposition)

설정/배경 정보를 대사로 전달할 때:
- **갈등 속에서** 자연스럽게 노출 (질문, 논쟁, 회상)
- 두 캐릭터가 이미 아는 내용을 서로에게 설명하는 대사는 피한다
- 독자가 모르는 것과 캐릭터가 모르는 것을 구분한다

### 대사 리듬

- 긴 문장과 짧은 문장을 교차하여 리듬을 만든다
- **침묵**(말줄임, 행동으로 대체)도 대사의 일부
- 감정이 고조될수록 문장이 짧아지는 패턴이 자연스럽다

## 톤/무드 가이드

| 장르 | 톤 키워드 | 묘사 특징 | 대사 특징 |
|------|----------|----------|----------|
| **판타지** | 장엄, 경이 | 스케일, 세계 디테일 | 격식체, 고유 용어 |
| **호러** | 불안, 공포 | 제한된 시야, 감각 왜곡 | 짧은 문장, 침묵 |
| **로맨스** | 설렘, 긴장 | 인물 중심, 감각적 | 서브텍스트, 말 끊기 |
| **SF** | 사변적, 냉정 | 기술/환경 디테일 | 전문 용어, 논리적 |
| **코미디** | 경쾌, 위트 | 과장, 타이밍 | 말장난, 반전, 콜백 |
| **일상물** | 따뜻, 소소 | 일상 디테일, 계절감 | 자연스러운 구어 |
| **느와르** | 비관, 건조 | 도시, 밤, 비 | 독백, 비유, 냉소 |
| **사극** | 장중, 비장 | 의례, 복식, 건축 | 존대/하대, 고어 |

장르를 혼합할 때는 **기본 톤 1개 + 보조 톤 1개**로 제한한다.

## 출력 형식: 씬 대본

```markdown
## 씬 [번호]: [씬 제목]

**장소**:
**시간**:
**등장인물**:
**씬 목표**: (이 씬이 스토리에서 수행하는 역할)

---

[지문] (배경, 분위기, 인물 동작)

**[캐릭터명]**
"대사"

[지문] (반응, 동작)

**[캐릭터명]**
"대사"

---

**연출 노트**: (톤, 음악, 카메라, 특수효과 — 필요 시)
```
