# 플롯/구조 설계 가이드

## 구조 프레임워크 선택

| 프레임워크 | 적합한 상황 |
|------------|------------|
| **3막 구조** | 영화, 단편, 범용 — 가장 보편적 |
| **기승전결** | 동양 서사, 단편 — 전환점 강조 |
| **영웅의 여정** | 판타지, 모험, 성장물 — 12단계 |
| **비선형 구조** | 미스터리, 실험적 서사 — 시간 역전, 다시점 |

사용자가 프레임워크를 모르면 장르/분위기 기반으로 추천한다.
복수 프레임워크 혼합 가능 (예: 3막 + 영웅의 여정).

### 3막 구조 체크리스트

1. **1막 (설정, ~25%)**: 일상 → 사건 발생 → 1차 전환점 (주인공이 행동을 결심)
2. **2막 (대립, ~50%)**: 시련 연속 → 중간점 (판이 뒤집힘) → 2차 전환점 (최대 위기)
3. **3막 (해결, ~25%)**: 클라이맥스 → 해소 → 새로운 일상

### 기승전결 체크리스트

1. **기**: 세계와 인물 소개, 분위기 설정
2. **승**: 사건 전개, 갈등 심화
3. **전**: 예상 밖 전환, 반전 또는 위기
4. **결**: 결말과 여운

### 영웅의 여정 (12단계 요약)

일상 세계 → 소명 → 소명 거부 → 멘토 만남 → 첫 관문 통과 → 시련/동맹/적 → 동굴 접근 → 시련(죽음과 재탄생) → 보상 → 귀환의 길 → 부활 → 영약과 귀환

> 12단계를 전부 사용할 필요 없다. 핵심 단계만 취사선택 가능.

## 갈등 설계

### 갈등 유형

- **인물 vs 인물**: 직접 대립 (주인공 vs 적대자)
- **인물 vs 사회**: 제도, 차별, 관습에 대한 저항
- **인물 vs 자연**: 재해, 생존, 환경
- **인물 vs 기술**: AI, 마법 폭주, 기술 부작용
- **인물 vs 자신**: 도덕적 딜레마, 트라우마, 정체성

### 갈등 계층화

- **메인 갈등** 1개: 전체 스토리를 관통하는 핵심 갈등
- **서브 갈등** 2-3개: 메인을 풍성하게 하거나 캐릭터를 드러내는 보조 갈등
- 서브 갈등이 메인 갈등의 해결을 방해하거나 촉진하면 좋다

## 긴장 곡선

- **리듬**: 긴장↑ → 소강 → 더 높이↑ (계단식 상승)
- **클리프행어**: 장/챕터 끝에 미해결 긴장을 남겨 다음을 유도
- **반전 배치**: 중간점 또는 2차 전환점에 배치가 효과적
- 페이싱이 빠른 장르(스릴러)는 소강 구간을 짧게, 느린 장르(일상물)는 길게

## 주제/테마

### 로그라인 작성법

> [주인공]이 [상황/사건]에 처하여 [목표]를 위해 [갈등/장애물]에 맞선다.

예시: "기억을 잃은 기사가 자신의 정체를 찾기 위해 왕국의 금지된 기록실에 잠입하지만, 과거의 자신이 왕국 최대의 적이었음을 알게 된다."

### 모티프 반복

- 핵심 이미지(빛/어둠, 거울, 새 등)를 반복하여 주제를 강화
- 대사 반복: 초반에 가벼운 의미 → 후반에 무거운 의미로 재등장 (콜백)
