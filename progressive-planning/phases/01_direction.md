## 방향 설정 (Layer 1)

방향 설정 → 구조 설계 → 세부 명세 순서로 진행한다.
각 단계의 구체적인 초안 포함 항목, 질문, 출력 형식은 스코프별 가이드를 따른다.

### 공통 진행 절차

방향 설정, 구조 설계, 세부 명세 모두 아래 절차를 동일하게 적용한다.

| | 단계 | 처리 |
|---|------|------|
| A | 초안 제시 | 스코프별 가이드의 "초안 포함 항목"에 따라 초안 작성. 도메인 가이드 로드 시 추가 절차 함께 실행 |
| A' | 대안 제시 | 구조적 분기 시 대안 2~3개를 강점/약점과 함께 제시, 권장안 표시. 집요: 모든 레이어 / 적당: 핵심만 / 빠르게: 직행 |
| B | 핵심 질문 | 스코프별 가이드의 "핵심 질문" 제시. `[핵심]` 질문은 건너뛸 수 없음 |
| C | 정제 및 확정 | 사용자 응답 반영 → 수정 사항 명시 → 확정 |
| D | 반복 | 구조 설계: 단계/영역별, 세부 명세: 구성요소별 **하나씩** A→(A')→B→C 반복 |
